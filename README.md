# vs-covid-19
        
コロナウイルスによる影響を少しでも軽減する何かができないかと思い、  
条件を満たす場合以下の対応を無償で行おうと考えています。  
なにかありましたらご連絡ください。

## 対象

コロナの影響で営業が厳しくなった方。

## 料金

ヒアリングのうえ妥当と思ったケースは無償

## 出来ること

* 対面せず資金をうけとる方法の相談
* 上記で相談した結果必要となったシステムの構築・運用
* システムの宣伝・流布のサポート

## なぜこのような形にするか

企業を通してしまうと本当に届けたい人に届くか不安。
または、既存のプラットフォームに登録してもらう等が大変なケースを想定し、
極力シンプルな形で送金出来るサービスをいっそ作れないかと検討しました。

が、上記を目指すと資金移動業等、クリアしなければならない課題が多く短時間での提供は難しい。今各種プラットフォーマーが立ち上げている支援プロジェクトを待ち、そういったサービスの紹介・運用を行った方が安全 & 現実的な気がしました。

そのため、あくまで個人ないし法人事業者の方が自身で運営するサイトに、
支援を募るための決済手段を公開する仕組みを作る手助けを無償にて行います。
